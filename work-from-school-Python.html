Image Processing Learn more on Github

In this project I used loops and nested loops to applies filters to an image.

  • "OneColor" returns an image containing only red, blue, or green, aspects of the original image.
  • "Negate" returns an image that is a conversion of the original image to photographic negative form.
  • "Scale" returns a scaled version of the image.
  • "Blur" returns a blurred version of the image.
