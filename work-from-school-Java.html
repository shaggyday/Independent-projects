Path Finder Learn more on Github

In this project I used the graph ADT and created an algorithm to find the shortest path between two Wikipedia links. The program reads from text files to create edges and vertices. Then, I used lists and queues to execute breadth-first search to find the shortest path.

Cloud Dreamer Learn more on Github

In this project I created a special tree class similar to a 2-3 tree that stores words from a given text. A node contains 2 words and their frequency in the text; a node can also have up to 3 children. The nodes are added in alphabetical order. Finally, Cloud Dreamer returns the words with highest frequency and prints a word cloud.

Maze Runner Learn more on Github

In this project I created a MazeSquare class to print a maze. I used lists and stacks to execute depth-first search for the program to solve the maze. It would keep traversing until it hits a dead end and then calls the stack's .pop method to trace back.
